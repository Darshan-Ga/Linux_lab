# 🚩 Bandit Level 0: The First Login

### 🎯 The Mission
This is where the real command-line work begins.
The challenge here was simple but foundational:
 figure out how to log into a remote server using SSH (Secure Shell) without relying on a graphical interface. 

---

### 💻 How I Did It
I needed to connect to the OverTheWire servers using a specific port. Here is the exact command I ran in my terminal to gain access:
```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
